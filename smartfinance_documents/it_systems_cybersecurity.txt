================================================================================
            4. CYBERSECURITY FRAMEWORK
================================================================================

4.1 SECURITY GOVERNANCE

- Board-approved Information Security Policy
- IT Strategy Committee oversight
- Chief Information Security Officer (reports to CTO)
- Annual security budget: ₹85 lakhs

4.2 SECURITY CONTROLS

NETWORK SECURITY:
- AWS WAF (Web Application Firewall)
- AWS Shield (DDoS protection)
- VPC with private subnets
- Network ACLs and Security Groups
- CloudFront CDN with TLS

APPLICATION SECURITY:
- OWASP Top 10 compliance
- Input validation and sanitization
- Parameterized queries (SQL injection prevention)
- Content Security Policy (XSS prevention)
- Rate limiting and throttling

DATA SECURITY:
- Encryption at rest: AES-256 (AWS KMS)
- Encryption in transit: TLS 1.2+
- Database encryption (RDS)
- Field-level encryption for PII
- Data masking in non-production

ACCESS CONTROL:
- Role-based access control (RBAC)
- Principle of least privilege
- Multi-factor authentication (MFA)
- SSO via Okta
- Privileged access management
- Quarterly access reviews

ENDPOINT SECURITY:
- CrowdStrike endpoint protection
- Mobile device management (MDM)
- USB port blocking
- DLP on endpoints
